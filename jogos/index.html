SOBRE O SITE

Esta página web UBIRATAN GAMES foi desenvolvida como uma das atividade da eletiva programação web. O objetivo é reunir uma lista de jogos educativos que possam ser utilizado pelos alunos e professores a fim de melhorar a aprendizagem dos estudantes.
